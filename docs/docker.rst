Docker
======

Le projet est conteneurisé avec Docker.

Commandes utiles
----------------

Construire l'image :

.. code-block:: bash

   make build

Lancer le conteneur :

.. code-block:: bash

   make run

Charger les données de démonstration :

.. code-block:: bash

   make loaddata

Afficher les logs :

.. code-block:: bash

   make logs

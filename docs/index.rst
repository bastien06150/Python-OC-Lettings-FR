Objectif du projet
------------------

Cette application Django permet d'afficher des profils utilisateurs et des biens immobiliers à louer.

Le projet inclut :

* une application Django refactorisée ;
* des tests automatisés ;
* une intégration Sentry ;
* une image Docker ;
* un déploiement Render ;
* une documentation Read The Docs.
